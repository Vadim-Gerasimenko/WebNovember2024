Задача "Работа с объектами"

Условие

  1. Создать массив объектов-стран
    • У страны есть название и список городов
      • У города есть название и численность населения
  2. Найти страны с максимальным количеством городов
  3. Получите объект с информацией по всем странам такого вида: ключ – название страны, значение – суммарная численность по стране

Требования

  • Оформить код в виде функций
  • Результаты печатать в консоль
  • Переменные и функции не должны попадать в глобальную область видимости (нужно использовать самовызывающиеся функции или namespace-ы)
  • Вместо циклов по массивам нужно по максимуму использовать функции ES5: filter, map, reduce, forEach и др.
  • forEach использовать только если другие функции не подходят по смыслу
  • Для переменных нужно использовать const и let
  • Использовать стрелочные функции там, где они подходят
