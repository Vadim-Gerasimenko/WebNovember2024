Задача "Классы"

  • Создать класс “Животное”, у которого должно быть поле “имя” и метод “говорить”
    • По умолчанию метод “говорить” должен печатать в консоль фразу “<имя> издает звук”
  • Сделать классы-наследники “Собака” и “Кошка”, в них нужно переопределить метод, чтобы вместо “издает звук” было “лает”/“мяукает”
  • Задачу нужно сделать в двух вариантах:
    1. Через функции-конструкторы и прототипы
    2. Через классы ES6
